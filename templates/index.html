Microphone: Use http://127.0.0.1:5000 (local) or your app URL (e.g. sst-jet.vercel.app). If Chrome doesn't ask for permission, go to chrome://settings/content/microphone → under "Allow" click Add → enter this site's URL (e.g. https://sst-jet.vercel.app or http://127.0.0.1:5000) → Add. Then reload and click "Allow microphone". If it says "No microphone detected", set a mic in Windows Settings → Sound → Input.

MP3, M4A, WebM are converted in the browser — no FFmpeg needed.

Recording uses your mic and transcribes in real time (no file upload needed; works without FFmpeg). If the browser didn't ask for mic access: Chrome → chrome://settings/content/microphone → Add http://127.0.0.1:5000 to Allow → reload.
